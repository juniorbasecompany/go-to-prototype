-------------

git pull
baixa o que o Diogo fez.

git status
mostra o que eu fiz e não foi submetido.

git add .
adiciona tudo para submetido.

git commit -m "faz...."
faz o commit no repositório local.

git push origin main
envia do repositório local para o repositório remoto.

-------------

git branch
mostra as branch's locais e em verde com asterísco a branch que estou.

git branch feature/add-new-json-structure
cria uma branch com o nome informado.

 git checkout feature/add-new-json-structure
 estou entrando na branch com o nome informado.
